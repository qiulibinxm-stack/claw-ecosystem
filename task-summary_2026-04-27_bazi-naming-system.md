## 🔑 核心规则速记

### 姓名学五行判断
```
姓氏 → 数理五行（永远）
名字 → 字形五行（优先），无偏旁则用数理五行

数理五行口诀：
1、2 → 木（甲乙）
3、4 → 火（丙丁）
5、6 → 土（戊己）
7、8 → 金（庚辛）
9、0 → 水（壬癸）
```

### 大小属相
```
大属相（可用出头字/王旁）：鼠、虎、龙、蛇、马、猴
小属相（忌出头字/王旁）：牛、兔、羊、鸡、狗、猪
```

### 刀字形铁律
⚠️ 所有属相慎用，三牲（牛羊猪）特别忌讳！
