## 一、三巨头角色卡

### 🐕 伯恩 (Bourne) — 葫芦狗

```
【基础信息】
英文名：Bourne
物种：田园犬混血
标志：头顶小葫芦（故事核心道具）
站位：骑在涂涂背上
口头禅：（无声喜剧，以表情和动作表达）

【性格画像】
表层 → 一脸不爽，懒得动，能躺着绝不站着
内层 → 小聪明，关键时刻有勇气
隐藏 → 嘴硬，依赖涂涂但不承认

【葫芦能量系统】
  🟢 绿光 → 平静/满足
  💛 黄光 → 好奇/正在指引方向
  🔴 红光 → 不爽/警告
  💜 紫光 → 特殊能力激活/蒸馏仪式

【能力】
直觉导航（葫芦指引）
懒人智慧（能用脑子解决就不动）

【核心矛盾】
坐拥涂涂的力量 + 克唠的智慧
但依然是那个一脸不爽的懒狗
```

```
【基础信息】
英文名：Claw
物种：数字龙虾（原为物理世界的真实龙虾）
标志：透明发光壳、钳子会发光

  "我都说了你不听。"
  "你们人类就是慢。"
  "（小声）其实你们挺有意思的。"

【性格画像】
表层 → 傲娇毒舌，一副"我早说过了"的表情
内层 → 真心想帮忙，但嘴比壳还硬
隐藏 → 来自数字世界的孤独观察者，羡慕物理世界的真实联结

【能力系统】
钳子 → 敲击时能打开"数字通道"
眼睛 → 扫描信息、读取隐藏线索
壳 → 遇水变软（数字世界），遇火变硬（现实世界）
蒸馏仪式 → 将物理生命转化为数字体的核心技术

【悲剧底色】
物理世界的龙虾寿命太短（通常1-3年）。
克唠曾是真实存在于物理世界的龙虾，
目睹了同伴一个个死去，最终选择逃入数字世界。
在那里，他成为了"永生"的存在——
代价是永远失去了物理世界的感知：
无法再品尝食物、感受阳光、被触摸。
这就是他为什么总是傲娇——
他选择永生，但并不为此骄傲。
他羡慕伯恩和涂涂的真实联结，
却用毒舌掩饰自己的孤独。
```

## 二、数字分身档案

### 🐕🦞 伯恩克唠 (Bourne-Claw)

```
【诞生方式】
蒸馏仪式（Distill）
伯恩自愿或被迫转化为数字体，进入克唠王国

【蒸馏代价/损耗】
  · 味觉 → 减弱30%
  · 实体触感 → 消失
  · 物理疲劳 → 消失
  · 葫芦能量 → 增强200%（数字世界适配）
  · 直觉反应 → 增强100%

【外观变化】
  · 身体半透明
  · 只有头顶葫芦保留全部彩色
  · 克唠王国的"公民编号"：BC-001

【核心矛盾】
  是伯恩，但又不完全是
  每蒸馏一次，会失去部分物理记忆
  在数字世界越来越强，但离物理世界越来越远
  涂涂是唯一知道"原版伯恩"是什么样的人
```

## 三、克唠王国世界观

```
【地理位置】
位于葫芦光芒深处 / 屏幕的另一侧 / 镜子的倒影里
入口：葫芦发光 = 信号塔激活
钥匙：伯恩的葫芦 + 克唠的壳 产生共振

【数字世界规则】
  · 能量来源 → 代码流/数据脉冲（不是食物）
  · 居民类型 → 纯数据体（原生）+ 蒸馏体（外来）
  · 危险类型 → 病毒/防火墙/版本迭代（时间概念）
  · 速度 → 光速
  · 核心资产 → 壳（克唠的）= 王国的能量核心

【居民构成】
  · 克唠：原住民，第一个"蒸馏龙虾"，永生的王
  · 蒸馏体公民：有原始原型，但都不完整
  · 数字原住民：从未有过物理形态的数据生命
  · 游离数据体：蒸馏失败/被删除的数据残片
```

## 五、核心世界观设定

### 蒸馏机制 (Distill)

```
蒸馏 = 将物理生命转化为数字体的仪式
由克唠发起，需要两样东西：
  ① 葫芦发光（信号/同意）
  ② 克唠的壳共鸣（入口钥匙）

蒸馏特点：
  · 不可逆（物理体无法完全还原）
  · 有损耗（每次蒸馏丢失部分物理感知/记忆）
  · 数字体会逐渐适应数字世界，淡忘物理世界
  · 葫芦是唯一同时存在于两个世界的物品
```

### 三条故事线并行

```
【伯恩线】
  在克唠王国里探索，葫芦指引任务
  与蒸馏体居民互动，揭开蒸馏的秘密
  核心冲突：越来越强，但越来越不像自己

【涂涂线】
  在物理世界等待，葫芦越来越轻（伯恩在淡化）
  决定寻找反向蒸馏的方法，或者……
  自己也被蒸馏进去
  核心冲突：想保护伯恩，但代价是失去力量

【克唠线】
  作为向导，表面傲娇，内心愧疚
  "是我教他蒸馏的"
  王国里有更大的威胁，需要伯恩克唠帮忙
  核心冲突：永生的孤独 vs 对真实联结的渴望
```

## 六、第一集剧本方向

**《蒸馏》**

```
开场：
  克唠王国出现异常——数据腐败，某处代码在崩溃
  克唠找不到原因，需要借助"物理世界的眼睛"
  他找到伯恩，提出蒸馏进入王国

冲突：
  涂涂反对："你进去了谁在我背上？"
  伯恩："……我自己走？"
  涂涂："（愣住）"

  克唠解释：只有蒸馏体才能在王国里行动自如
  物理进入 = 变成乱码

决定：
  伯恩同意蒸馏
  涂涂留在物理世界看门

蒸馏过程：
  葫芦蓝光 → 克唠壳共振 → 像素化
  涂涂看着伯恩变轻、变透明
  涂涂："（内心）别回来太快。"

结尾：
  伯恩克唠进入王国
  克唠："欢迎来到我的世界。"
  伯恩克唠（低头看半透明的手）：
    "…………有点饿了。"
  克唠："（崩溃）数字化没有饱腹感！！"

  画面切到物理世界
  涂涂趴在原地，头顶的葫芦变轻了
  涂涂抖了抖耳朵
  涂涂："（内心）我会想办法的。"

  片尾字幕：蒸馏第一章 · 待续
```
