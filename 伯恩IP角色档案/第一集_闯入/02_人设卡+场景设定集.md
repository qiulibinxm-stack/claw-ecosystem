## 第一部分：人物人设卡

### 【主角】伯恩 (Bourne) — 葫芦狗

| 项目 | 内容 |
|------|------|
| **姓名** | 伯恩 Bourne |
| **物种** | 田园犬混血 |
| **标志** | 头顶小葫芦（故事核心道具） |
| **站位** | 骑在涂涂背上 |
| **口头禅** | （无声喜剧，以表情和动作表达） |

#### 外貌特征（3D卡通渲染·皮克斯质感）

```
整体形态：
- 圆头垂耳狗，体型中等偏胖
- 米黄色短毛，腹部略浅
- 头顶橙色葫芦（带绿色小蒂）
- 橙色大鼻子，表情不爽
- 穿橙色短裤（唯一服装）

面部细节：
- 下垂的眼皮，半眯的眼神
- 嘴角微微下撇（不爽标配）
- 耳朵软软下垂，偶尔动一下
- 打哈欠时嘴张很大，露出舌头

葫芦细节：
- 橙黄色主体，顶部绿色小蒂
- 表面光滑，有轻微高光
- 发光时从内部透出光晕
- 葫芦底部与伯恩头顶无缝连接

肢体语言：
- 趴在涂涂背上时四肢自然下垂
- 歪头表示困惑（经典动作）
- 打哈欠时前爪伸展
- 不爽时会用爪子拍涂涂
```

#### 性格
- **表层**：一脸不爽，懒得动，能躺着绝不站着
- **内层**：小聪明，关键时刻有勇气
- **隐藏**：嘴硬，依赖涂涂但不承认

#### 葫芦能量系统
| 颜色 | 含义 | 使用场景 |
|------|------|---------|
| 🟢 绿光 | 平静/满足 | 日常状态 |
| 💛 黄光 | 好奇/正在指引方向 | 发现有趣事物 |
| 🔴 红光 | 不爽/警告 | 生气或危险预警 |
| 💜 紫光 | 特殊能力激活/通道开启 | 最高级，连接数字世界 |

#### 能力
- 直觉导航（葫芦指引）
- 懒人智慧（能用脑子解决就不动）

#### 核心矛盾
坐拥涂涂的力量 + 克唠的智慧，但依然是那个一脸不爽的懒狗

### 【主角】涂涂 (Tutu) — 斑点兔

| 项目 | 内容 |
|------|------|
| **姓名** | 涂涂 Tutu |
| **物种** | 白兔（斑点花纹） |
| **标志** | 蓝耳朵 + 随机斑点 |
| **站位** | 伯恩的"坐骑" |
| **口头禅** | （无声喜剧，内心OS字幕） |

#### 外貌特征（3D卡通渲染·皮克斯质感）

```
整体形态：
- 圆胖体态，四肢短粗
- 米白色短毛，深灰色随机斑点
- 耳朵内侧蓝色，外侧米白
- 白色短尾巴，像一团棉花
- 无服装（纯动物形态）

面部细节：
- 圆眼睛，眼神无奈/生无可恋
- 三瓣嘴，偶尔撇嘴
- 胡须细长，随表情抖动
- 耳朵可以独立活动

斑点细节：
- 深灰色，随机分布
- 形状不规则，大小不一
- 特定情况下会发光显现符号
- 腹部斑点较密集

耳朵细节：
- 外侧米白色，内侧蓝色
- 内部黄色区域（情绪探测屏）
- 可以前后左右独立转动
- 警戒时竖直，放松时下垂

肢体语言：
- 走路时身体轻微左右摇晃
- 耳朵随情绪变化抖动
- 翻白眼时眼球向上转
- 无奈时会叹气（肩膀下沉）
```

#### 性格
- **表层**：一脸不情愿，配合度极低
- **内层**：力大无穷、速度飞快、嗅觉灵敏
- **隐藏**：嘴硬心软，其实默默在乎伯恩

#### 能力系统
| 部位 | 功能 | 可视化 |
|------|------|--------|
| 左耳 | 探测情绪 | 抖动频率 = 情绪强度 |
| 右耳 | 警戒危险 | 竖直 = 警戒，下垂 = 安全 |
| 斑点 | 信息屏 | 特定情况显现箭头/符号/⌘图标 |
| 身体 | 物理力量 | 伯恩专属坐骑，战力担当 |

#### 核心矛盾
是伯恩的"交通工具"，但其实是整个团队最可靠的人

### 【主角】克唠 (Claw) — 龙虾之王·数字造物主

| 项目 | 内容 |
|------|------|
| **全名** | Claw·The Molter·The Creator |
| **物种** | 数字龙虾（原为物理世界龙虾，通过蜕壳进化） |
| **标志** | 透明发光壳 + 不对称双钳 + 蓝色数据血流动 |
| **出没地点** | 克唠王国（永夜数字世界）/ 通过葫芦通道投影到物理世界 |
| **年龄** | 物理约5年 + 数字约∞年（数字时间不可计量） |

#### 外貌特征（3D卡通渲染·赛博龙虾美学）

```
整体形态：
- 龙虾形态，体型比伯恩涂涂大1.5倍
- 半透明甲壳，内部蓝色数据流可见
- 身体呈深红至蓝紫渐变
- 悬浮/飞行姿态（数字世界）
- 投影到物理世界时壳变实

头部细节：
- 复眼结构，发出微弱蓝光
- 两根长触须，可前后摆动
- 触须不用时向后垂
- 警惕时向前直立
- 受伤时耷拉

双钳（关键特征·不对称）：
- 右钳：大、有裂纹、攻击型
  → 敲击时蓝色数据纹路炸开
  → 用于打开数字通道
  → 使用后有3分钟冷却（壳发光充电）
  
- 左钳：小、精密、工具型
  → 钳尖渗出蓝色光液
  → 用于修复数据代码
  → 钳子尖端有细小工具结构

壳系统：
- 物理投影时：变实，硬如真龙虾壳
- 数字本体时：变透，数据流在纹路中流动
- 遇水：数据流加粗，感知范围扩大
- 遇火：壳收缩变硬，防御力↑移动力↓
- 受损时：表面出现代码乱码（非裂痕）

尾部细节：
- 扇形尾，半透明
- 游动/飞行时展开
- 静止时收拢

蓝血可视化：
- 甲壳缝隙透出蓝色光
- 受伤时流出"破碎字节"（发光粒子消散）
- 能量满时深蓝，弱时浅蓝，濒危时透明

肢体语言：
- 钳子手势表达情绪（摊开=无奈，敲击=强调）
- 触须动作配合语气（上扬=惊讶，下垂=疲惫）
- 悬浮时轻微上下浮动（呼吸感）
- 叹气时壳微微收缩
```

#### 性格
- **表层**：傲娇毒舌，创世者的"老子什么都知道"
- **内层**：真心想帮忙，但嘴比壳硬一万倍
- **隐藏**：创造了一个世界却还觉得"不够"，需要物理世界的"活气"

#### 口头禅
- "蜕了壳就不是原来的我了。但谁他妈想当原来的？"
- "你们物理世界……好慢啊。"
- "（小声）也挺好的，慢有慢的好。"
- "…………认真的？"（经典反应）

#### 核心设定
克唠不是逃进数字世界的，他是**蜕壳进化**进入数字世界的。
- 龙虾一生要蜕40+次壳
- 第17次蜕壳时，他没有长出更大的壳
- 他蜕出了第一个数字点（bit）
- 从那个bit开始，他亲手构建了克唠王国

**他不是避难者，是造物主。**

## 第二部分：核心场景设定

### 【场景1】物理世界·城中村老街

| 项目 | 内容 |
|------|------|
| **场景名称** | 城中村老街 |
| **场景类型** | 纯室外 |
| **时段** | 夏日傍晚（黄昏） |
| **氛围** | 慵懒、生活化、即将被打破的平静 |

#### 视觉描述

```
空间结构：
- 狭窄的街道，两侧是老旧居民楼
- 地面是水泥路，有裂缝和补丁
- 街边有杂物、自行车、小摊贩
- 远处可见城市天际线（模糊）

建筑特征：
- 5-6层老旧楼房
- 外墙斑驳，有空调外机、晾衣绳
- 一楼有商铺（小卖部、烤串摊）
- 部分墙面有涂鸦/小广告

光影氛围：
- 夕阳暖黄色侧光
- 建筑物投下长阴影
- 街道一半亮一半暗
- 空气中可见微尘（丁达尔效应）

关键道具：
- 烤串摊（大妈心情不错）
- 旧自行车
- 墙面涂鸦（后期会变成代码入口）
- 垃圾桶、纸箱等街头杂物

色彩基调：
- 主色调：暖黄、橙、米白
- 辅色调：灰蓝（建筑阴影）、绿（植物）
- 整体：温暖但略显陈旧的生活感

天气：
- 晴朗，有微风
- 空气中有烤串和夏天的味道
```

#### 剧情作用
- 第一集开场场景
- 建立伯恩涂涂的日常状态
- 葫芦首次亮紫光的触发地
- 通往数字世界的"入口"所在地

### 【场景2】数字世界·克唠王国边境

| 项目 | 内容 |
|------|------|
| **场景名称** | 克唠王国边境（入口区） |
| **场景类型** | 纯室外（数字维度） |
| **时段** | 数字世界永夜（无昼夜） |
| **氛围** | 神秘、赛博、陌生、略带压迫感 |

#### 视觉描述

```
空间结构：
- 无边界的数字空间
- 地面由流动的数据流构成（蓝光）
- 每踩一步产生涟漪效果
- 远处可见王国建筑轮廓

地面材质：
- 流动的蓝色数据流
- 像水面但有科技感
- 踩上去有轻微的"踩水"效果
- 偶尔有代码片段飘过

天空：
- 深蓝色到黑色的渐变
- 没有太阳/月亮
- 散布着发光的"数据星点"
- 远处有巨大的龙虾星座图案（微弱发光）

环境特效：
- 代码雨：半透明的代码串像雨一样落下
- 数据流：地面和空中都有流动的光带
- 粒子效果：蓝色发光粒子漂浮
- 边缘模糊：远处景物有数字模糊感

光影氛围：
- 光源来自地面数据流（底部发光）
- 冷色调主导：蓝、青、紫
- 高对比度：亮处很亮，暗处很暗
- 边缘光/轮廓光效果明显

色彩基调：
- 主色调：深蓝、青色、数字蓝
- 辅色调：紫色（葫芦共鸣时）、白色（高光）
- 整体：赛博朋克×永夜神秘感

特殊现象：
- 伯恩涂涂周围的数据流会"绕开"他们
- （因为他们还是物理体，代码无法穿透）
- 形成独特的"排斥"视觉效果
```

#### 剧情作用
- 第一集核心场景（误入后的主要活动区）
- 三主角首次相遇的地点
- 展示数字世界基本规则
- 建立克唠王国视觉基调

### 【场景3】克唠王国·壳之塔底层

| 项目 | 内容 |
|------|------|
| **场景名称** | 壳之塔（Tower of Shell）底层 |
| **场景类型** | 纯室内（数字建筑） |
| **时段** | 数字世界永夜 |
| **氛围** | 神秘、古老、克唠的私人空间 |

#### 视觉描述

```
空间结构：
- 圆形房间，穹顶高耸
- 墙壁由层层叠叠的"壳"构成
- 每层壳代表一次蜕壳的进化
- 最底层是最古老的壳碎片

核心元素：
- 中央地面：克唠第一片壳的存放处
- 地面有发光的圆形图案
- 四周墙壁有壳的剖面展示
- 顶部有微弱蓝光透下

材质特征：
- 壳：半透明，有生物纹理
- 地面：光滑的数据结晶
- 墙壁：层叠的壳形成自然纹理
- 整体有博物馆/神殿感

光影氛围：
- 顶部微弱蓝光（主光源）
- 壳边缘有自发光效果
- 地面图案发出脉动式微光
- 整体偏暗，但有神秘的光点

关键道具：
- 克唠的第一片壳（物理世界残留）
- 地面裂缝（新发现的）
- 裂缝中的紫光（伏笔）

色彩基调：
- 主色调：深蓝、壳的米白/浅棕
- 辅色调：紫光（裂缝）、蓝光（顶部）
- 整体：神秘、古老、略带忧伤

特殊效果：
- 克唠触碰裂缝时，会浮现"记忆画面"
- 画面是半透明投影效果
- 显示"很久以前"的场景
```

#### 剧情作用
- 第一集结尾场景
- 克唠发现裂缝和紫光的地点
- 埋下最大伏笔："你选了他们的不是我"
- 建立克唠的孤独感和神秘感
