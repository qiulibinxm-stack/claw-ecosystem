## 第一条 合作宗旨与基本原则

### 1.1 合作宗旨
双方建立平等互利的业务合作关系，充分发挥甲方的平台资源优势和乙方的专业技术优势，共同为患者提供优质的中医诊疗服务，实现合作共赢。

### 1.2 基本原则
1. **平等自愿**：双方地位平等，基于自愿原则开展合作；
2. **独立经营**：双方各自独立承担经营责任，不存在隶属或管理关系；
3. **风险自担**：各自承担因自身行为产生的法律责任和经济风险；
4. **诚信守约**：双方应诚实守信，严格履行本协议约定的义务。

## 第二条 合作内容与方式

### 2.1 合作内容
乙方借助甲方提供的经营场所及相关设施，开展以下中医诊疗服务：
1. 中医诊断（望、闻、问、切）；
2. 中药方剂开具与调理指导；
3. 针灸、推拿、拔罐等中医适宜技术；
4. 中医健康咨询与养生指导；
5. 双方协商确定的其他中医相关业务。

### 2.2 合作方式
1. **坐诊模式**：乙方定期或不定期到甲方场所坐诊，为患者提供诊疗服务；
2. **预约模式**：患者通过甲方平台预约乙方诊疗时间；
3. **会诊模式**：甲方可邀请乙方参与疑难病例会诊；
4. **其他模式：** _________________________（双方可另行约定）。

## 第三条 收益分配

### 3.1 分配方式
双方采用**固定比例分成**方式，无底薪。

### 3.2 分成比例
甲方每月向乙方支付甲方门店**当月营业总额的16%**作为乙方合作收益。

**重要说明**：
- 该16%比例基于甲方门店**整体营业额**计算，非乙方个人诊疗业绩；
- 营业额以甲方实际收款金额为准（含现金、刷卡、移动支付、医保结算等所有渠道）。

### 3.3 成本费用说明
双方确认，上述16%分成比例为**净分配比例**，已综合考虑以下因素：
1. 乙方借助甲方平台、场所、患者资源开展诊疗服务；
2. 甲方承担场地租金、水电、基础设备维护、日常运营等成本；
3. 双方各自承担自身的税费、保险等费用。

### 3.4 结算周期与方式
1. **结算周期**：按月结算，每月____日前结算上月收益；
2. **结算方式**：银行转账至乙方指定账户；
3. **对账方式**：
   - 甲方每月提供门店营业额明细表；
   - 乙方有权查阅相关财务凭证；
   - 乙方应在收到明细后5个工作日内核对确认，逾期未提出异议视为认可；
4. **税务处理**：乙方应依法自行申报缴纳个人所得税，甲方有代扣代缴义务的除外。

## 第四条 双方权利与义务

### 4.1 甲方权利
1. 对乙方的诊疗行为进行合规性监督（不涉及专业技术判断）；
2. 要求乙方遵守甲方的基本管理制度（仅涉及场所秩序、安全、卫生等）；
3. 按照约定比例获得合作收益；
4. 使用乙方的专业资质进行合法宣传（需事先征得乙方同意）；
5. 对严重违反本协议或法律法规的行为，有权终止合作。

### 4.2 甲方义务
1. 提供符合卫生标准的诊疗场所和必要的基础设施；
2. 提供合法的医疗机构资质，配合乙方办理多点执业备案（如需要）；
3. 负责患者的预约登记、收费结算、档案管理等基础服务；
4. 协助乙方进行个人品牌宣传和患者引流；
5. 按约定及时足额向乙方支付合作收益；
6. 为乙方提供必要的行政支持（如开具发票、医保对接等）；
7. 每月向乙方提供真实、完整的门店营业额明细。

### 4.3 乙方权利
1. 自主决定诊疗方案、用药处方和技术操作（在合法合规范围内）；
2. 按照约定获得合作收益；
3. 使用甲方提供的场所、设备和患者资源；
4. 要求甲方提供必要的配合和支持；
5. 查阅甲方门店营业额相关财务凭证；
6. 对甲方的违规行为提出异议并要求改正。

### 4.4 乙方义务
1. 严格遵守《中华人民共和国中医药法》《医疗机构管理条例》等法律法规；
2. 按照中医诊疗规范开展服务，确保医疗质量和安全；
3. 遵守甲方的基本管理制度（场所秩序、安全、卫生、保密等）；
4. 妥善保管和使用甲方的设施设备，如有损坏应赔偿；
5. 维护甲方的商誉，不得做出有损甲方形象的行为；
6. 配合甲方进行必要的宣传、培训和学术交流活动。

## 第五条 执业资质与责任承担

### 5.1 资质要求
1. 乙方应确保其《中医（专长）医师执业证书》或《中医类别医师资格证书》真实、合法、有效；
2. 乙方应在执业证书规定的执业范围内开展诊疗活动；
3. 如乙方执业地点变更或证书到期，应及时告知甲方并办理相关手续；
4. 甲方应确保其《医疗机构执业许可证》合法有效，诊疗科目包含乙方执业范围。

### 5.2 医疗责任承担
1. **诊疗行为责任**：乙方独立承担因其诊疗行为（诊断、处方、操作等）产生的医疗责任；
2. **场所设施责任**：甲方独立承担因场所设施缺陷、管理不善导致的患者损害责任；
3. **混合责任**：因双方共同过错导致的责任，按过错比例分担；
4. **保险建议**：双方均应自行购买医疗责任保险或执业保险，降低风险。

### 5.3 医疗纠纷处理
1. 发生医疗纠纷时，双方应积极配合调查，如实提供相关资料；
2. 因乙方诊疗行为引起的纠纷，由乙方主导处理，甲方提供必要协助；
3. 处理纠纷产生的费用（赔偿、诉讼费、律师费等），按责任归属承担；
4. 双方应建立医疗纠纷应急预案，及时妥善处理突发事件。

## 第八条 协议的变更、解除与终止

### 8.1 协议变更
本协议的任何变更或补充，须经双方协商一致，并以书面形式确认。

### 8.2 协议解除
**任何一方提前解除协议的，应提前30日书面通知对方。**

**甲方可立即解除协议的情形：**
1. 乙方提供虚假资质证明或执业证书失效；
2. 乙方严重违反诊疗规范，造成重大医疗事故；
3. 乙方私自收费、收受回扣，损害甲方利益；
4. 乙方严重违反甲方基本管理制度，经警告后拒不改正。

**乙方可立即解除协议的情形：**
1. 甲方《医疗机构执业许可证》被吊销或注销；
2. 甲方长期拖欠合作收益，经催告后仍不支付；
3. 甲方要求乙方从事违法违规的诊疗活动；
4. 甲方严重损害乙方声誉或合法权益；
5. 甲方提供的营业额数据严重不实，经核实后拒不改正。

### 8.3 协议终止后的处理
1. 协议终止后，双方应在10个工作日内完成财务结算；
2. 乙方应将甲方提供的场所、设备、资料等归还甲方；
3. 双方应妥善处理在诊患者的后续诊疗安排；
4. 保密义务和知识产权条款继续有效。

## 补充说明

### 关于16%营业额分成的特别说明

本协议约定的16%收益分配方式为：**乙方按甲方门店当月总营业额的16%获取合作收益**，而非乙方个人诊疗业绩的16%。

**举例说明**：
- 若甲方门店当月营业额为10万元，则乙方当月收益为：10万 × 16% = 1.6万元
- 该分成与乙方个人接诊量、开药金额无直接关联

**优势**：
- 乙方收益与门店整体经营状况挂钩，有利于双方共同促进门店发展
- 简化核算流程，避免个人业绩统计争议

**风险提示**：
- 乙方应关注门店整体经营状况
- 建议甲方每月提供真实、完整的营业额明细供乙方核对
