## 四、技术选型

### 前端
- **框架**: React 18 + TypeScript
- **UI库**: TailwindCSS + shadcn/ui
- **状态管理**: Zustand
- **路由**: React Router v6
- **构建**: Vite
- **图标**: Lucide React
- **动画**: Framer Motion

### 后端
- **框架**: Next.js 14 (App Router)
- **数据库**: PostgreSQL (用户数据) + Redis (会话)
- **文件存储**: S3/Cloudflare R2
- **实时通信**: Socket.io (虾聊)

### 部署
- **前端**: Vercel / Cloudflare Pages
- **后端**: Railway / Vercel
- **数据库**: Supabase
