# 五行协作看板系统 - 产品蓝图

## 🎯 核心愿景

将五行团队（6个Agent）打造成一个**可视化协作系统**，让你能：
- 看到任务实时运行状态
- 与每个虾单独沟通
- 直接投喂多模态内容（图/视频/音频/文档）

## 🔧 技术实现路径

### 方案A：纯Web界面（推荐）

**第一步：创建看板前端**
- 使用frontend-ui-ux skill设计UI
- React SPA，通过Gateway API与后端通信
- 核心功能：
  - 任务看板（看板式拖拽）
  - 运行监控（实时状态流）
  - 消耗统计（图表展示）
  - Agent对话面板（点击进入）

**第二步：多模态上传**
- 拖拽上传 → 存储到workspace → 通知对应Agent
- 支持格式：图片(jpg/png/gif)、视频(mp4/webm)、音频(mp3/wav)、文档(pdf/docx)

**第三步：实时通信**
- Gateway HTTP API + 轮询/SSE
- 状态推送：任务进度、Agent活动、消耗更新

## 📋 MVP功能清单（第一阶段）

### 核心功能
1. **任务看板**
   - 创建任务 → 分配给五行Agent
   - 状态流转：待办→进行中→已完成
   - 任务详情：描述、附件、子任务

2. **Agent对话面板**
   - 点击Agent头像进入专属对话
   - 显示该Agent当前任务列表
   - 直接发送消息/文件

3. **运行监控**
   - 实时显示Agent活动日志
   - 当前执行任务高亮
   - 错误/警告提示

4. **多模态上传**
   - 拖拽上传文件
   - 选择目标Agent
   - 添加说明文字

### 数据存储
- 任务数据：JSON文件（workspace/projects/wuxing-dashboard/data/tasks.json）
- 会话记录：OpenClaw自带LCM（lossless-claw）
- 上传文件：workspace/uploads/
