## 一、命与名的关系

### 核心理论

> "名者，命也！"

- **命**：先天命（出生刹那接受宇宙能量）+ 后天命（生命的使命，由自己选择）
- **名**：给生命赋予的称谓，是生命的证明
- **字**：对名的解释和补充，对名的解析、放大、深入刻画

### 名与命相通
- 名与命是直通车
- 名以正体，字以表德

### "道"的本义
- 道字 = 十字路口 + 首（动脑）+ 止（脚步）
- 在十字路口要往前行正道，动脑筋才能找到自己的道

### "德"的本义
- 德字 = 十字路口 + 大眼睛 + 指南针
- 有德之人看问题精准，方向永远朝前
- 心无旁骛视为德

## 二、命名解码核心法门 — 二五八法则

> 这是姓名学的核心方法论，贯穿所有起名实践

| 维度 | 内容 | 含义 |
|------|------|------|
| **二** | 阴阳 | 汉字的阴阳属性（笔画奇偶/字义刚柔） |
| **五** | 五行 | 汉字的五行属性（木火土金水） |
| **八** | 八卦 | 汉字的八卦对应（乾坤震巽坎离艮兑） |

### 二五八法则应用
1. 分析八字 → 确定用神五行
2. 按五行选字 → 二五八法则过滤
3. 阴阳平衡 → 名字整体阴阳协调
4. 八卦对应 → 与命局八卦位配合

## 三、四种起名法

| 方法 | 原理 | 优点 | 缺点 |
|------|------|------|------|
| **八字起名法** | 根据八字用神选字 | 最精准 | 需要八字基础 |
| **生肖起名法** | 根据生肖属相选字 | 简单易用 | 精度有限 |
| **五行起名法** | 根据五行生克选字 | 体系完整 | 需五行知识 |
| **数理起名法** | 根据笔画数理选字 | 有数理依据 | 历史最短 |

### 四种起名法的历史发展
- 最早：八字起名法（先秦已有）
- 其次：生肖起名法（汉代发展）
- 再后：五行起名法（唐宋成熟）
- 最晚：数理起名法（日本传入，近代发展）

## 四、飞刀神断法 — 三大铁律

### 铁律一：生肖属相取名
- 根据出生年份的生肖
- 每个生肖有喜用字和忌用字
- 生肖与地支对应，影响字的选择

### 铁律二：字形结构判断
- 左右结构 vs 上下结构
- 字形本身有符咒之力
- 字形影响运势走向

### 铁律三：字义五行判断
- 字义本身的五行属性
- 与用神五行的配合
- 忌用与忌神五行相同的字

## 五、男性取名底层逻辑

### 男名讲究
1. **阳刚之气** → 用阳刚字义（刚、强、勇）
2. **事业取向** → 用官禄相关字（达、成、立）
3. **家族传承** → 字辈排列
4. **用神补救** → 根据八字补用神

### 男名避雷
- ❌ 过于柔弱之字
- ❌ 与忌神五行相同之字
- ❌ 笔画过繁之字
- ❌ 谐音不吉之字

## 七、起名禁忌

| 禁忌 | 原因 | 后果 |
|------|------|------|
| 忌用与忌神五行相同的字 | 加重命局失衡 | 运势更差 |
| 忌谐音不吉 | 字音是咒语 | 潜意识负暗示 |
| 忌字形破败 | 字形是符咒 | 影响气场 |
| 忌过于极端之字 | 阴阳失衡 | 运势极端 |
| 忌与长辈同字 | 犯讳 | 不敬 |

## 八、汉字的阴阳五行

### 汉字功能
1. **字形是符咒** → 视觉能量
2. **字音是咒语** → 听觉能量

### 汉字阴阳判断
- 笔画奇数 → 阳
- 笔画偶数 → 阴
- 字义刚强 → 阳
- 字义柔弱 → 阴

### 汉字五行判断
| 方法 | 原理 | 举例 |
|------|------|------|
| 字义法 | 字义本身五行 | 林→木、炎→火、山→土 |
| 偏旁法 | 偏旁部首五行 | 木旁→木、火旁→火 |
| 笔画法 | 笔画数理五行 | 尾数1/2→木、3/4→火 |
| 音韵法 | 发音五行 | 角→木、徵→火 |

## 十、康熙字典知识库速查

### 核心规则
```
姓氏 → 永远用【数理五行】
名字 → 优先【字形五行】，无明显偏旁则用【数理五行】
```

### 数理五行口诀
```
1、2 → 木（甲乙）
3、4 → 火（丙丁）
5、6 → 土（戊己）
7、8 → 金（庚辛）
9、0 → 水（壬癸）
```

### 大小属相速记
```
大属相（可用出头字/王旁）：鼠、虎、龙、蛇、马、猴
小属相（忌出头字/王旁）：牛、兔、羊、鸡、狗、猪
```
