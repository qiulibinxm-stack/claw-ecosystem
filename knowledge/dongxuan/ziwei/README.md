## 四化详解

### 四化含义（四季对应）

| 四化 | 季节 | 五行 | 核心含义 |
|------|------|------|----------|
| **化科** | 春耕 | 木 | 善缘、解厄、贵人、功名、珍惜、才艺 |
| **化权** | 夏耘 | 火 | 成就、管制、霸性、自负、掌权、原则 |
| **化禄** | 秋收 | 金 | 人缘、财禄、情缘、才艺、享受 |
| **化忌** | 冬藏 | 水 | 亏欠、道义、凶祸、粘住、变动、失去、获得 |

### 四化分类
- **生年四化**：出生年天干决定，一生不变
- **宫干四化**：各宫位天干飞出的四化，动态变化

### 宫干四化·吉化线

| 线位 | 吉化含义 |
|------|----------|
| 命迁线 | 工作施展才华，社会地位 |
| 父疾线 | 前途光明，升迁顺利 |
| 财福线 | 薪水/创业财运 |
| 子田线 | 合伙运、下属关系 |
| 官夫线 | 工作成功、工作姻缘 |
| 兄友线 | 人际关系、朋友运 |

### 宫干四化·化忌冲

| 化忌冲 | 含义 |
|--------|------|
| 化忌命宫冲迁移 | 无法施展才华 |
| 化忌迁移冲命 | 工作多变动 |
| 化忌父母冲疾厄 | 易受工伤 |
| 化忌疾厄冲父母 | 与领导不合 |
| 化忌财帛冲福德 | 工作烦心 |
| 化忌福德冲财帛 | 因工作损财 |
| 化忌子女冲田宅 | 需离家工作 |
| 化忌田宅冲子女 | 不利合伙 |
| 化忌夫妻冲官禄 | 工作不稳 |
| 官禄自化忌 | 比别人付出更多 |
| 化忌兄友线 | 人际关系差 |

### 四化入宫规则
- **化入**：化入命宫/官禄/财帛/田宅 → 禄赚钱/权掌权/科贵人/忌守财
- **化出**：化入其他宫位 → 禄守不住/权争权/科贵人隐/忌损财

---

## 解盘核心公式

**A宫四化入B宫冲（照）C宫**
- "引动"：将某一宫位引动
- "串联"：将两个宫位串联
- 例：命宫化禄入夫妻 → 命宫引动夫妻宫
