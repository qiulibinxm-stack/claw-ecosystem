## 核心原则

> "宅者，人之本，人因宅而立，宅因人得有。人宅相扶，道天地居。若安，则家代昌吉。"

### 风水本质
- **风水 = 环境磁场能量**
- 好风水：藏风纳气，聚气聚福，聚财
- 坏风水：如人体经络不通，影响健康财运整体运势

### 关键认知
> ⚠️ 风水布局的调整**没有普适通用方法**，千人千面，万宅万运。一套房子的风水可能适合别人，但不一定适合自己。故风水调整**必须根据主人的八字喜忌、流年运势、所求气运来专门调整**。

## 九宫布局法

### 操作方法
1. 将自家户型分为九宫方位
2. 找出主人运势对应的方位
3. 看九宫飞星位落在哪个房间
4. 根据对应方位进行风水布局调整

## 商业风水

### 职场风水调整技巧
- 座位方位：背靠实墙（有靠山）
- 办公桌朝向：面向用神方位
- 避免正对大门/厕所
- 左青龙右白虎布局

### 店铺/工厂风水
- 门面开阔 → 聚气
- 收银台位置 → 财位
- 仓库布局 → 生机位
- 动线设计 → 气流畅通

### 风水催旺财运
1. 找到财位（结合主人八字）
2. 财位放置催财物品
3. 保持财位干净整洁
4. 配合流年飞星调整

## 阴阳宅理论

### 阳宅（活人住宅）
- 重视采光、通风、格局方正
- 人宅相扶，宅安人吉

### 阴宅（墓地）
- 重视龙脉、水法、砂法
- 阴宅影响后代运势（传统观点）

## 7天转运秘法

1. 结合流年运势找出不利因素
2. 化解霉运：调整居住环境
3. 增强旺运：催旺用神方位
