# 八字命理·用神法篇 — 四大用神法详解

> 来源：少嵛《真神论命》配套资料07-15 + 文档资料第12-16/31节

---

## 核心原则

> 用神 = 对日干产生好作用的五行
> 找用神 = 找到八字最需要的五行来平衡命局

### 用神/喜神/忌神/仇神关系

```
用神 ← 喜神（帮助用神）
忌神 ← 仇神（帮助忌神）

例：甲木日主，身弱
  用神 = 壬癸水（生我）
  喜神 = 庚辛金（生水·前提是命中有水）
  忌神 = 庚辛金（克我·无水时金克木）
  仇神 = 戊己土（生金克水）

⚠️ 辩证：如木已极旺，水再增则"水多木漂"，水反成忌神
```

---

## 一、扶抑法 — 最基础的用神法

### 原理
- 日元强 → 抑之（用克泄耗）
- 日元弱 → 扶之（用生扶帮）

### 判断日元强弱的要素

| 因素 | 强日元 | 弱日元 |
|------|--------|--------|
| 月令 | 得令（同五行或生我） | 失令 |
| 地支 | 有根（通根） | 无根 |
| 天干 | 有比劫印绶帮扶 | 无帮扶或被克泄 |
| 地支合局 | 三合局增强 | 冲合减力 |

### 扶抑法取用神口诀
- 身强：用官杀克制、食伤泄秀、财星消耗
- 身弱：用印绶生扶、比劫帮身

---

## 二、病药法 — 对症下药

### 原理
- 八字有"病"（某种五行过旺或过弱），需要找"药"来治
- 用神 = 治病的药

### 常见"病"与"药"

| 病 | 表现 | 药（用神） |
|----|------|-----------|
| 官杀太旺克身 | 压力大、受压制 | 食伤制杀 / 印化杀 |
| 财星太旺耗身 | 为财所困、劳碌 | 比劫帮身 / 印生身 |
| 食伤太旺泄身 | 才华过露、精力不足 | 印绶制伤 / 财泄食伤 |
| 印绶太旺埋身 | 依赖、懒散 | 财破印 / 比劫分印 |

---

## 三、通关法 — 桥梁和解

### 原理
- 两行交战（如金木相克），需要中间五行来"通关"
- 通关 = 找到能沟通两者的五行

### 通关组合

| 交战双方 | 通关五行 | 原理 |
|---------|---------|------|
| 金克木 | 水 | 金生水、水生木 |
| 木克土 | 火 | 木生火、火生土 |
| 水克火 | 木 | 水生木、木生火 |
| 火克金 | 土 | 火生土、土生金 |
| 土克水 | 金 | 土生金、金生水 |

### 通关法的深层理解
- 五行生化 → 阴阳平衡 → 能量流通
- 不是简单的"和稀泥"，而是让能量有序流转
- 通关成则格局高，通关败则两败俱伤

## 四、调候法 — 寒暖燥湿

### 原理
- 命局过寒需暖、过热需凉、过燥需润、过湿需燥
- 调候用神优先于扶抑用神

### 四库与五行旺衰
| 四库 | 五行 | 特性 |
|------|------|------|
| 辰 | 湿土（水库） | 润、含蓄 |
| 戌 | 燥土（火库） | 燥、热烈 |
| 丑 | 寒土（金库） | 寒、收藏 |
| 未 | 暖土（木库） | 暖、生发 |

### 调候法实战要点
1. 夏月生人 → 需水调候（解炎）
2. 冬月生人 → 需火调候（解寒）
3. 辰戌月生 → 需察燥湿
4. 调候用神往往比扶抑用神更关键

## 五、三眼法 — 日元旺衰综合判断

| 步骤 | 方法 | 核心动作 |
|------|------|----------|
| **第一眼·劈刀法** | 看日元与月令 | 得令否？月令是帮还是克？ |
| **第二眼·打勾法** | 看日元自身 | 生扶多还是克泄多？ |
| **第三眼·循环法** | 全局循环 | 五行力量对比，谁强谁弱？ |

### 三眼法实例（乙木日主）

```
第一眼：乙木生在午月 → 不是很旺
第二眼：乙木生伤官丙火，日支酉金克乙木 → 日元能量弱
第三眼：年干壬水通根，可克火生木 → 用神为水、木

结论：乙木身弱，用神水木
```

## 六、格局法 — 命局层次判断

### 五类命局

| 格局 | 特征 | 人生倾向 |
|------|------|----------|
| **格** | 用神有力、格局清 | 大富大贵 |
| **局** | 用神可用、格局中 | 中上命运 |
| **平** | 用神无力、格局平 | 平凡人生 |
| **卑** | 用神受伤、格局低 | 运势低迷 |
| **凶** | 无用神可取 | 灾祸频仍 |

### 格局法要点
1. 原局透出的十神决定格局类型
2. 凶神有制→格局高（如伤官配印）
3. 凶神无制→格局低
4. 格局可以随大运流年变化

### 高等格局示例：伤官配印
- 条件：伤官透出+正印克制伤官
- 特点：才华横溢+有约束，上等八字
- 反面：伤官无制→叛逆一切，四海为家
