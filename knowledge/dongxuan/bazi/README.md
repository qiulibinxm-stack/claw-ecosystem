> 来源：林白亲传弟子班课 · 少嵛《真神论命》系列
> 五行归属：水·智 — 深沉、洞察、流动

---

## 体系架构

```
八字命理
├── 基础层：阴阳→五行→天干→地支→十神
├── 方法层：扶抑法/病药法/通关法/调候法（四大用神法）
├── 判断层：三眼法（劈刀/打勾/循环）+ 格局法
├── 实战层：财运/婚姻/事业/学业/健康
└── 补救层：用神定位→五行调和→流年时机
```

## 关键概念速查

### 五行生克
- **相生**：木→火→土→金→水→木（顺生）
- **相克**：木→土→水→火→金→木（隔克）
- **旺衰**：当令者旺、令生者相、生令者休、克令者囚、令克者死

### 天干（十干）
| 天干 | 五行 | 阴阳 | 合化 |
|------|------|------|------|
| 甲 | 木 | 阳 | 甲己化土 |
| 乙 | 木 | 阴 | 乙庚化金 |
| 丙 | 火 | 阳 | 丙辛化水 |
| 丁 | 火 | 阴 | 丁壬化木 |
| 戊 | 土 | 阳 | 戊癸化火 |
| 己 | 土 | 阴 | - |
| 庚 | 金 | 阳 | - |
| 辛 | 金 | 阴 | - |
| 壬 | 水 | 阳 | 壬丙相冲 |
| 癸 | 水 | 阴 | 癸丁相冲 |

| 十神 | 关系 | 核心含义 |
|------|------|----------|
| 正印 | 生我 | 母亲、学业、贵人 |
| 偏印（枭神） | 生我 | 偏才、孤独、多疑 |
| 比肩 | 同我 | 兄弟、竞争、独立 |
| 劫财 | 同我 | 争夺、冲动、合作 |
| 食神 | 我生 | 才华、福气、子女 |
| 伤官 | 我生 | 叛逆、才华、口舌 |
| 正财 | 我克 | 妻子、稳定收入 |
| 偏财 | 我克 | 父亲、意外之财 |
| 正官 | 克我 | 事业、丈夫、律法 |
| 七杀（偏官） | 克我 | 压力、权威、风险 |

### 四大用神法

| 方法 | 核心逻辑 | 适用场景 |
|------|----------|----------|
| **扶抑法** | 日元强则抑、弱则扶 | 最基础的用神法 |
| **病药法** | 八字有病需找药 | 命局有偏颇时 |
| **通关法** | 五行生化、阴阳平衡 | 两行交战时 |
| **调候法** | 寒暖燥湿要调和 | 极寒极热命局 |

### 三眼法（判断日元旺衰）

| 眼 | 方法 | 要点 |
|----|------|------|
| 第一眼 | 劈刀法 | 看日元与月令的关系（得令否） |
| 第二眼 | 打勾法 | 看日元自身强弱（生扶vs克泄） |
| 第三眼 | 循环法 | 全局循环看五行力量对比 |

### 用神/喜神/忌神/仇神

| 概念 | 定义 | 举例（甲木日主） |
|------|------|------------------|
| **用神** | 对日干产生好作用 | 壬癸水（生我） |
| **喜神** | 帮助用神的能量 | 金（生水） |
| **忌神** | 对日干产生坏作用 | 庚辛金（克我） |
| **仇神** | 帮助忌神的五行 | 土（生金） |

> ⚠️ 辩证思维：如木已很旺，水再多则"水多木漂"，水反成忌神
