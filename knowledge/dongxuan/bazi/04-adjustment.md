## 三、贵人运补救

### 天乙贵人查法

| 日干 | 贵人地支 |
|------|---------|
| 甲戊 | 丑、未 |
| 乙己 | 子、申 |
| 丙丁 | 亥、酉 |
| 壬癸 | 卯、巳 |
| 庚辛 | 寅、午 |

### 增强贵人运的方法
1. **命中有天乙贵人** → 先天贵人缘厚
2. **食神得用** → 以才华吸引贵人
3. **用神方位** → 往用神方位发展，自然遇贵人
4. **流年遇贵人** → 贵人年抓住机会
5. **名字补救** → 姓名中补用神五行（见姓名学）

## 四、流年时机选择

### 最佳行动时机
- 流年天干地支与用神一致 → 赚钱年/转运年
- 流年合化用神 → 加持年
- 流年冲克忌神 → 转运年
- 大运+流年同时助用神 → 黄金十年

### 避险时机
- 流年天干地支与忌神一致 → 需谨慎
- 流年冲克用神 → 守成为主
- 大运走忌神运 → 保守策略

## 五、姓名补救（衔接姓名学）

> "名者，命也！名是给生命赋予的一个称谓，它是你生命的证明。"

### 姓名与八字的关系
- 名与命是相通的，名与命是直通车
- 名以正体，字以表德
- 好名字 = 用神五行的加持

### 姓名补救原则
1. 根据用神五行选字
2. 避免忌神五行用字
3. 注意字形（符咒）、字音（咒语）
4. 汉字本身有阴阳五行属性
