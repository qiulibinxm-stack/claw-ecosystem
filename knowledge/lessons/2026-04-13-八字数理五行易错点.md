# 八字取名：姓名五行判断易错点

> 记录时间：2026-04-13

## 踩坑记录

### 坑1：数理五行法则记错
**错误**：把5画归为木，把6画归为火。

**正确法则（必须按这个记）**：
- 1、2 画 → 木（单数甲木，双数乙木）
- 3、4 画 → 火（单数丙火，双数丁火）
- 5、6 画 → 土（单数戊土，双数己土）
- 7、8 画 → 金（单数庚金，双数辛金）
- 9、0 画 → 水（单数壬水，双数癸水）

**口诀**：木1-2，火3-4，土5-6，金7-8，水9-0

### 坑2：混淆数理五行和部首五行的适用范围
**错误**：拿到一个名字，不区分姓和名，一律用数理五行。

**正确流程**：
1. 姓氏 → 100% 数理五行（笔画）
2. 名字 → 优先部首五行；无部首可定五行 → 数理五行

**口诀**：姓用笔画数，名先看部首，部首没有再看笔画。

### 坑3：误判"佐恩"的五行
**错误过程**：佐=7画→金，恩=10画→水 → 金水组合。后又改为恩=心→火→金火组合。

**正确分析**：
- 佐（姓）→ 数理：7画→庚金 ✅
- 恩（名）→ 部首=心→心不在五行偏旁体系中 → 回退数理：10画→癸水

**结果**：庚金（姓）+ 癸水（名）→ **金水组合**（金生水，相生相旺）

**关键**：心不属于火部首！火部首只有火和日。不要自行拓展偏旁五行对应关系。

### 教训总结
1. 先问是"姓"还是"名"，这是判断的第一前提
2. 名优先部首，只有无部首才用数理五行
3. 数理五行按笔画个位数，不能套用偏旁五行的4=木、6=火等印象
4. 偏旁五行严格按师承标准：木(木艹)、火(火日)、土(土山+孙)、金(钅釒)、水(冫氵+雨冬雪泉+政)
5. 心不属于火！不要自行拓展偏旁对应关系
