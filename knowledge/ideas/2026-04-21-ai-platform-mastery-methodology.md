# AI平台掌握方法论（元方法论）

> **一句话摘要**：从"普通人玩转OpenClaw"提炼出的通用方法论——如何系统掌握任何一个AI工具平台。

---

## 核心洞察

"普通人如何玩转OpenClaw"这个手册本身揭示了一个可迁移的方法论：

> **掌握任何AI平台 = 找控制面板 → 拆核心模块 → 注入灵魂 → 扩展技能 → 排错兜底**

这不是OpenClaw专属的，而是适用于Cursor、Dify、Coze、n8n、LangChain等所有AI工具平台的通用框架。

---

## 五步掌握法

### 🔍 第一步：找控制面板（5分钟）

**原理**：任何复杂系统都有一个"总控"入口，找到它就掌握了全局。

| 平台 | 控制面板 |
|------|---------|
| OpenClaw/QClaw | `openclaw.json` 配置文件 |
| Cursor | Settings → Models → Rules |
| Dify | 左侧菜单"设置" |
| Coze | 工作空间设置 |
| n8n | Settings → Credentials |

**关键问题**：
1. 配置在哪里？（文件/UI/环境变量）
2. 改完是否自动生效？（热重载 vs 重启）
3. 有没有备份机制？（改坏了能回退吗）

---

### 🧩 第二步：拆核心模块（15分钟）

**原理**：复杂系统 = 少数核心模块 + 大量可选模块。80%的功能来自20%的模块。

**拆解方法**：
1. 列出所有顶级配置项/菜单项
2. 按重要性分3层：
   - **Layer 1 能不能用**：认证、模型接入、渠道连接
   - **Layer 2 好不好用**：技能/插件、工作流、工具权限
   - **Layer 3 高级定制**：路由、会话管理、网关配置
3. 只聚焦Layer 1，其他用默认值

**常见错误**：
- ❌ 试图一次性理解所有模块
- ❌ 在Layer 3上花时间而Layer 1还没配好
- ✅ 先让系统跑起来，再逐步优化

---

### 👤 第三步：注入灵魂（10分钟）

**原理**：配置是骨架，人格是灵魂。没有灵魂的AI = 高级计算器。

**灵魂三要素**：
1. **身份**（WHO）：这个AI是谁？什么性格？什么风格？
2. **记忆**（MEMORY）：它记得什么？怎么学习？
3. **规则**（RULES）：什么能做？什么不能做？怎么做决策？

| 平台 | 身份 | 记忆 | 规则 |
|------|------|------|------|
| OpenClaw | SOUL.md | MEMORY.md | AGENTS.md |
| Cursor | .cursorrules | Chat History | Project Rules |
| Dify | System Prompt | Variable | Workflow |
| Coze | 人设与回复逻辑 | 变量/数据库 | 工作流约束 |

**核心洞察**：大多数人只做到配置（骨架），没有做到灵魂注入，所以AI永远只是个"工具"而不是"助手"。

---

### 🔌 第四步：扩展技能（持续迭代）

**原理**：基础配置决定了"能用"，技能扩展决定了"好用"。

**技能扩展三路径**：
1. **内置技能**：平台自带，开关控制
2. **社区技能**：Marketplace/Hub安装
3. **自建技能**：按需开发，最灵活

**关键认知**：
- 技能 > 配置。5分钟配好的配置，不如一个精心选择的技能包
- 技能的组合 > 单个技能。真正的威力来自技能之间的协同
- 定时/自动化 > 手动。从"被动回答"到"主动服务"的质变

---

### 🛡️ 第五步：排错兜底（随时）

**原理**：没有不翻车的系统，只有不翻车的准备。

**排错三板斧**：
1. **诊断命令**：`openclaw doctor` / `--version` / `status`
2. **日志追踪**：看错误信息，不要猜
3. **最小复现**：砍到最简配置，逐步加回

**安全底线**：
- 新手先从最小权限开始，不要全开
- 有备份才敢动手
- `trash` > `rm`（可恢复 > 不可逆）

---

## 方法论的元认知

这个五步法本身就是一个**可迭代的框架**：

```
第1轮：找控制面板 → 让系统跑起来（30分钟）
第2轮：拆核心模块 → 理解系统架构（1小时）
第3轮：注入灵魂 → 让AI成为助手（30分钟）
第4轮：扩展技能 → 解决实际问题（持续）
第5轮：排错兜底 → 建立安全边界（随时）
```

每轮都有明确的完成标志，不是"学完再动手"，而是"动手中学"。
