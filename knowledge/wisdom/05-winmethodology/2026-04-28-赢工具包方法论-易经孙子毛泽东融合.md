## 二、核心方法论：商业制胜八字诀

从30个案例中提炼出**8个核心盈利策略**：

### ① 借（借力打力）
**定义**：用别人的资源、资源、渠道，零成本启动
**案例**：
- 案例1：有机食品超市借力大餐饮（借餐饮客流、场地、信任背书）
- 案例8：整合美容院+整形医院（借别人的客户、场地、项目）
- 案例6：利用别人服装店实现自己品牌扩张

**毛泽东对应**：「自己动手，丰衣足食」+「统一战线」
> 不从零开始建资源，而是发现已有资源，重新组合

**孙子兵法对应**：「故善战者，求之于势」

---

### ② 拆（化整为零）
**定义**：把重资产拆成轻资产，把一次性投入变成持续收入
**案例**：
- 案例19：重资产定制家具厂转型轻资产（卖方案不卖工厂）
- 案例26：重资产餐饮获得海量客户+轻资产成长
- 案例4：自有品牌阿胶膏设计多赢战略（拆分成代理/合伙/联营）

**毛泽东对应**：「伤其十指，不如断其一指」
> 集中优势兵力，单点突破，而不是全面铺开

---

### ③ 锁（锁定客户）
**定义**：设计收费站/管道，让客户持续付费
**案例**：
- 案例3：电子乐器厂打造自动盈利收费站商业模式
- 案例23：健康养生项目快速盈利并打造收费站
- 案例24：医疗仪器厂打造自动盈利的收费站

**易经对应**：谦卦䷎ — 谦虚让人依赖你，而不是离开你
> 收费站的核心不是一次性收费，而是持续、自动、被动收入

---

### ④ 变（转型升级）
**定义**：从卖产品→卖服务→卖模式→卖系统，逐级升级
**案例**：
- 案例2：连锁餐饮开业前锁定海量客户（卖股权+卖客户+卖系统）
- 案例22：婚纱摄影修图公司从人力密集型转型系统盈利
- 案例7：负债地块解除包袱并快速持续盈利

**易经对应**：乾卦䷀ — 自强不息，持续进化
> 模式不是一成不变的，每个阶段需要不同的打法

---

### ⑤ 合（产业链整合）
**定义**：打通上下游，消除中间环节利润损耗
**案例**：
- 案例8：整合美容院+整形医院零成本重组产业链
- 案例28：汽车维修连锁轻资产扩张（整合配件供应链）
- 案例29：零成本控制重资产矿山上市公司（金融+产业整合）

**孙子兵法对应**：「知彼知己，百战不殆」
> 先搞清楚整个产业链的利润分布，找到可以整合的节点

---

### ⑥ 占（占领心智）
**定义**：在细分领域做到第一，占领用户认知
**案例**：
- 案例17：近视防控机构发展招商加盟（占领品类认知）
- 案例5：特色旅游/农业/文旅轻资产快速启动
- 案例13：颈椎痛的中医合作快速盈利（占领专科定位）

**毛泽东对应**：「枪杆子里面出政权」→ 流量即权力
> 占领用户心智就是占领市场

---

### ⑦ 圈（圈层运营）
**定义**：围绕特定人群深度运营，创造高粘性社区
**案例**：
- 案例27：儿童健康产品整合资源快速推广（妈妈圈层）
- 案例25：高端医疗养生会所开发高端客户
- 案例9：新型光电美容设备快速占领市场（美容院老板圈层）

**易经对应**：坤卦䷁ — 厚德载物，积累信任资产
> 圈层的本质是信任的积累，不是简单的建群

---

### ⑧ 收（收割变现阶段）
**定义**：当积累到临界点，设计高效变现路径
**案例**：
- 案例21：两千亩茶园把未来收益提前变现
- 案例30：电瓶车经销商零投入整合产业链并打造自有品牌
- 案例14：健康产业城零成本众筹启动

**毛泽东对应**：「最后的胜利，往往在于再坚持一下的努力之中」
> 时机未到不收割，时机到了果断收割

## 三、孙子兵法五事 × 商业评估框架

「经之以五事，校之以计」——开战之前先做庙算。

### 五事（战略评估）

| 五事 | 商业含义 | 一人公司自检问题 |
|------|----------|----------------|
| **道** | 使命/用户痛点 | 用户为什么需要你？有多痛？ |
| **天** | 时机/趋势 | 现在是这个方向的时机吗？ |
| **地** | 市场位置 | 你在哪里打？细分市场在哪？ |
| **将** | 创始人能力 | 你最擅长什么？能坚持多久？ |
| **法** | 执行系统 | 你的商业模式能复制吗？ |

### 七计（竞争分析）

```
主孰有道 → 谁能赢得用户信任？
将孰有能 → 谁的能力更强、更专注？
天地孰得 → 谁更把握趋势和时机？
法令孰行 → 谁的执行力更强？
兵众孰强 → 谁积累的资源更多？
士卒孰练 → 谁的专业度更高？
赏罚孰明 → 谁的激励更有效？
```

### 虚实篇应用（一人公司竞争优势）

**大公司的「实」（优势）**：
- 资源多 → 决策慢、人均产出低
- 团队大 → 协同差、内部损耗大
- 品牌强 → 船大难掉头、无法定制

**一人公司的「虚」（优势）**：
- 灵活 → 快速响应、小步迭代
- 专注 → 单点深度、专业度超越
- 定制 → 个性化服务、大公司做不了

**六字诀**：
- 敌进我退 → 大公司进入你的领域 → 聚焦更细分市场
- 敌驻我扰 → 竞争对手稳固 → 持续内容渗透
- 敌疲我打 → 对手精力分散 → 集中突破一点
- 敌退我追 → 对手撤退 → 抢占空白市场
- 我专为一 → 集中时间精力做一件事
- 敌分为十 → 让竞争对手多线作战

## 四、毛泽东思想 × 一人公司执行系统

### 原则一：集中优势兵力（聚焦）
> 「伤其十指，不如断其一指。」

**执行方法**：
- 每月只做**一件事**
- 每周只攻**一个渠道**
- 每天只服务**一类用户**

### 原则二：战略上藐视，战术上重视（心态）
> 「在战略上我们要藐视一切敌人，在战术上我们要重视一切敌人。」

**执行方法**：
- 想大目标（年入百万、行业第一）
- 做小行动（每一个用户反馈都认真对待）

### 原则三：持久战（节奏）
> 「最后的胜利，往往在于再坚持一下的努力之中。」

**执行方法**：
- 设置最低行动线：每天至少做一件与目标有关的事
- 建立复利思维：今天的积累，未来指数级爆发
- 接受非线性：成长是阶梯式，不是线性的

### 原则四：农村包围城市（路径）
> 毛泽东：「从边缘切入，不要正面硬刚。」

**执行方法**：
- 先做细分垂直第一
- 再横向扩展
- 不是从大市场切入，而是从大公司看不上的小市场切入

### 毛泽东语录 × 商业翻译

| 原文 | 商业翻译 |
|------|----------|
| 「星星之火，可以燎原」 | 从最小MVP开始，最终可以改变行业 |
| 「自己动手，丰衣足食」 | 不过度依赖外部资源，用现有条件起步 |
| 「没有调查没有发言权」 | 先做用户调研，再做产品设计 |
| 「枪杆子里面出政权」 | 流量即权力，获得流量就是获得主动权 |
| 「打得赢就打，打不赢就跑」 | 快速验证，不行就换，不要死磕 |
| 「一切反动派都是纸老虎」 | 大公司看起来可怕，找到软肋就能以弱胜强 |

## 五、易经决策框架（商业版）

当你面临商业决策时，用这个框架：

```
1. 我现在处于什么「卦」？
   （起步=乾 / 探索=屯 / 积累=需 / 扩张=巽 / 守成=谦）

2. 这个决策是「元亨利贞」还是「凶」？
   （是否符合天道？是否符合长期价值？）

3. 变爻在哪里？
   （什么因素变化会导致结果完全不同？）

4. 我的行动是否符合卦象的时位？
   （时机对吗？位置对吗？）

5. 孙子兵法五事检验：
   道（用户痛点）天（时机）地（市场）将（能力）法（系统）
```

## 七、一人为核心行动系统

### 每日行动（最低行动线）
1. **得一**：获得一个精准用户/潜在客户
2. **输出一**：发布一条有价值的内容
3. **思考一**：记录一个商业洞察

### 每周检视（五事评估）
1. 道：本周有没有更理解用户痛点？
2. 天：本周有没有把握趋势的机会？
3. 地：本周有没有在细分市场建立优势？
4. 将：本周有没有提升核心能力？
5. 法：本周商业模式有没有优化空间？

### 每月进化（卦象升级）
```
起步期（乾）→ 找方向，定战略
探索期（屯）→ 突破从0到1
积累期（需）→ 积累第一批100个用户
扩张期（巽）→ 找到可复制模式
守成期（谦）→ 建立护城河
```
