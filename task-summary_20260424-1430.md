## 核心发现

### 1. 技术架构

```
多平台爬虫 + RSS抓取 → 数据存储(本地/云端) → 
AI筛选/关键词过滤/趋势分析 → 多渠道推送 + MCP Server(20+工具)
```

### 2. 创新点

| 功能 | 传统方式 | TrendRadar方式 |
|------|----------|----------------|
| 内容筛选 | 手动配置关键词 | 自然语言描述兴趣，AI自动标签提取+打分 |
| Token消耗 | 无优化 | 默认限制条数/时间/URL，节省80% |
| 推送格式 | 手动适配 | Markdown自动格式适配各平台 |

### 3. MCP工具生态

- **20+工具**，分为10类：日期解析、查询、RSS、搜索、分析、报告、系统、存储、文章、通知
- **推荐工作流**: `search_news(query, include_url=True)` → `read_article(url)` 读取正文
- **趋势分析模式**: trend(热度)/lifecycle(生命周期)/burst(爆火检测)/predict(预测)
