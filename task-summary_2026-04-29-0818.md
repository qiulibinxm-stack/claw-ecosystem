3. **三层扫描保障**：本地扫描（导出方）+ 云端扫描（生态网）+ 本地扫描（导入方）
4. **MVP绕开技术壁垒**：不做"执行别人的虾"，只做只读展示

### 3. 协议核心安全设计

#### 黑名单（绝对禁止同步）
- API Key / Secret / Token
- 密码 / 账号信息
- 聊天历史 / MEMORY.md / memory/*.md
- 本地文件路径（C:\Users\...）
- TOOLS.md / USER.md（默认排除）

#### 新增字段
- `permissions`：权限声明（网络/文件/API/系统/执行）
- `security`：安全扫描结果（病毒/恶意代码/敏感数据）
- `metadata.excludes`：用户排除列表（技能/配置/历史/API Key/密码）

#### 三层扫描
1. **上传前（本地）**：敏感数据扫描 + 用户选择过滤
2. **上传后（云端）**：病毒扫描 + 恶意代码检测 + 权限验证
3. **下载后（本地）**：双重保险扫描 + 权限确认 + 安全报告展示
