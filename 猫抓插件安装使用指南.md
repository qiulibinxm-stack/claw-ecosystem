## 二、安装步骤（2分钟搞定）

### 方法A：.crx 文件安装（推荐）

1. **下载插件文件**
   - 访问上面的下载地址
   - 下载得到 `.crx` 文件（如 `猫抓_2.6.3.crx`）

2. **打开 Chrome 扩展管理页面**
   - 地址栏输入：`chrome://extensions/`
   - 回车进入

3. **开启开发者模式**
   - 页面右上角找到「开发者模式」开关
   - **打开它**（必须！）

4. **拖放安装**
   - 把下载的 `.crx` 文件从文件夹
   - **直接拖到** Chrome 扩展管理页面
   - 松开鼠标，点击「添加扩展程序」

5. **安装成功**
   - 浏览器右上角出现 🐱 猫爪图标
   - 点击图标即可使用

### 方法B：文件夹格式安装（备用）

如果 .crx 安装失败：

1. 把下载的 `.crx` 文件**重命名为 `.zip`**
2. 解压得到一个文件夹
3. 打开 `chrome://extensions/`
4. 开启「开发者模式」
5. 点击「加载已解压的扩展程序」
6. 选择刚才解压的文件夹

## 三、抓取网课视频（以"不直有见解"为例）

### 步骤 1：打开网课页面
- 用 Chrome 访问你的网课链接
- 开始播放视频

### 步骤 2：点击猫抓图标
- 点击浏览器右上角的 🐱 猫爪图标
- 插件会自动嗅探页面上的媒体资源

### 步骤 3：找到 M3U8 链接
- 在弹出的列表中，找到类型为 **M3U8** 的链接
- 通常文件名包含 `index.m3u8`、`playlist.m3u8` 或 `.m3u8`

### 步骤 4：下载视频
**方式 A - 直接下载（推荐）**：
- 点击 M3U8 链接后面的「下载」按钮
- 猫抓会自动合并所有分片，下载完整 MP4

**方式 B - 复制链接用 yt-dlp**：
- 点击「复制链接」
- 打开 PowerShell，执行：
```bash
yt-dlp "你复制的M3U8链接"
```

**方式 C - 小窗播放确认**：
- 点击「播放」按钮，先确认是不是你要的视频
- 确认后再下载

## 四、常见问题

### Q1：猫抓图标不显示？
- 点击 Chrome 右上角的「拼图」图标 🧩
- 找到猫抓，点击「固定」

### Q2：抓不到 M3U8？
- **刷新页面**后再试
- 确保视频正在播放
- 有些网站需要登录后才能抓取

### Q3：下载的视频无法播放？
- 可能是加密 M3U8，猫抓无法解密
- 尝试用「复制链接」+ yt-dlp 下载
- 或者用 OBS 录屏（保底方案）

### Q4：下载速度慢？
- M3U8 是分段下载，需要合并
- 大视频请耐心等待
- 或用 yt-dlp 配合 `--concurrent-fragments 10` 加速

## 五、yt-dlp + 猫抓 组合拳（高级）

如果猫抓直接下载失败，用 yt-dlp 接力：

```bash
# 基础下载
yt-dlp "M3U8链接"

# 指定输出路径
yt-dlp -o "D:/网课/%(title)s.%(ext)s" "M3U8链接"

# 多线程加速下载
yt-dlp --concurrent-fragments 10 "M3U8链接"

# 只下载音频（如果只需要讲课声音）
yt-dlp -x --audio-format mp3 "M3U8链接"
```
