## 🎯 备份策略（CEO确认版）

### 核心逻辑
- ✅ **每日增量备份**：知识 + 记忆 + 技能 + 对话 + 行为
- ✅ **时间轴标记**：每天一个独立恢复点
- ✅ **崩坏回滚**：回到崩坏前一天 → 恢复正常

### 安全机制
- 恢复前自动预归档当前状态到 `_pre_restore/`
- MD5校验确保备份完整性
- 时间轴索引防止搞混恢复点

## 🚀 快速使用

### 每日备份（自动运行，也可手动）
```bash
cd C:\Users\Administrator\.qclaw\workspace-agent-cf443017\五行备份系统\scripts
python 五行增量备份.py              # 实际备份
python 五行增量备份.py --dry-run    # 预览今日变更
```

### 恢复（崩坏时使用）
```bash
python 五行恢复工具.py --list       # 列出所有恢复点
python 五行恢复工具.py --date 2026-04-18  # 回滚到指定日期
python 五行恢复工具.py --latest    # 恢复到最新
python 五行恢复工具.py --element 水       # 只恢复水元素相关文件
```

### 开源情报（每日自动采集）
```bash
python 五行开源情报.py               # 采集今日情报
python 五行开源情报.py --brief      # 只显示摘要
```
