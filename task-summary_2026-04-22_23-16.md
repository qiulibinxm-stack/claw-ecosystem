# 硬件配置分析与 Gemma 模型选型

**用户配置**:
- CPU: Intel i7-6820HQ (4核8线程, 2015年Skylake)
- 内存: 32GB（使用量31.9GB，几乎满载）
- GPU: NVIDIA Quadro M2000M (4GB GDDR5显存, Maxwell架构)
- 核显: Intel HD 530

**关键瓶颈**: GPU显存仅4GB

**可运行的 Gemma 版本**:
1. ✅ Gemma 2 2B (FP16 ~4GB显存，4bit ~1.5GB) — 推荐，Ollama一键安装
2. ⚠️ Gemma 4 E4B (需4bit量化 ~3GB显存，或CPU offload)

**推荐方案**:
- 方案1: Gemma 2 2B — 最稳妥，`ollama run gemma2:2b`
- 方案2: Gemma 4 E4B + 4bit量化 — 需Python环境
- 方案3: CPU offload — 显存不够时借用内存

**注意事项**:
- 内存使用量已达31.9GB，运行前需释放空间
- Quadro M2000M为2015年产品，性能有限
- 注意笔记本散热
